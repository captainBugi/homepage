# Owen's Portfolio Project

이 프로젝트는 풀스택 개발자 **권오**의 개인 포트폴리오 웹사이트입니다. 정적 HTML과 CSS만을 사용하여 빠르고 가벼우며, 현대적인 디자인 시스템을 갖추고 있습니다.

## 🚀 프로젝트 개요

- **목적:** 개인 이력 및 기술 스택, 수행 프로젝트를 소개하는 포트폴리오 사이트
- **특징:** 
  - 모던하고 깔끔한 UI (Cursor-inspired design)
  - 반응형 웹 디자인 지원 (Mobile, Tablet, Desktop)
  - CSS Custom Properties를 활용한 체계적인 디자인 시스템
  - 외부 라이브러리 없이 순수 HTML/CSS 기반의 최적화된 성능

## 🛠️ 기술 스택

- **Markup:** HTML5 (시맨틱 태그 활용)
- **Styling:** CSS3 (Variables, Flexbox, Grid, Animations)
- **Typography:** Inter Font Family
- **Icons/Images:** SVG, PNG

## 📂 주요 구조

- `index.html`: 전체 웹사이트의 콘텐츠 구조 정의
- `index.css`: 전역 스타일링, 레이아웃 및 애니메이션 정의
- `profile.svg`, `profile.png`: 프로필 이미지 자산

## 💻 실행 방법

이 프로젝트는 별도의 빌드 도구 없이 즉시 실행 가능합니다.

1. 이 저장소를 클론합니다.
2. `index.html` 파일을 웹 브라우저로 엽니다.
3. (추천) VS Code의 **Live Server** 확장을 사용하여 실시간 미리보기를 활성화합니다.
